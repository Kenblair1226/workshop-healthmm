09Lab 02

Lab 02 · Code 重構

🧩

Domain / Application / Infrastructure / API 四層拆分。

♻️

去重複

10模型選擇

Pro Tip · 模型選擇 Model Selection

對的任務，配對的模型

⚡

快速 / 輕量

補全、樣板、簡單重構 — 反應快、成本低。

🧠

深度推理

跨檔案重構、架構設計、難解 bug — 用強推理模型。

🎯

切換時機

同一任務卡關 ≥2 次，升級模型再嘗試。

重點：同一個重構任務，比較不同模型的輸出與成本，選對工具。

15Lab 05 · 🎤 Demo

Lab 05 · 容器化 → Azure App Service · 🎤 講師示範

一個 Dockerfile，部署上雲

  • 用 Copilot 產生 multi-stage Dockerfile
  • 本機 docker run 驗證映像檔
  • 推送至 Azure Container Registry (ACR)
  • 部署至 App Service for Containers
az acr build -r <acr> -t his:v1 .
az webapp create --name his-web \
  --deployment-container-image-name \
  <acr>.azurecr.io/his:v1

最快上雲路徑：適合單體 / 內部系統，維運成本低。
